<h3>5. Desain Antarmuka Pengguna</h3>

Antarmuka pengguna (user interface) merupakan bentuk tampilan grafis yang berhubungan langsung dengan pengguna (user). Antarmuka pengguna berfungsi berfungsi untuk menghubungkan antara pengguna dengan sistem, sehingga sistem tersebut bisa digunakan.

<h3>5.1 Antarmuka Deskripsi</h3>

Deskripsi Antarmuka Pengguna merupakan penjelasan singkan dari tampilan grafis yang berhubungan langsung dengan pengguna(user).

<h3>5.1.1 Deskripsi modul User</h3>

<b>Halaman Utama</b><br/>
dapat melihat menu bar yang berisi Home, About, Data, dan Login. kemudian di bawah menu bar terdapat slide image yang berisi kegiatan di ganesha operation tersebut
dibawahnya ada icon kalender, Fasilitas, dan persyaratan untuk membuat akun tersebut. di bawah icon-icon terdapat sistem informasi tentang ganesha operation.
![](http://i63.tinypic.com/15iap13.jpg) <br/>

<b>Halaman About</b><br/>
halaman About dapat mellihat informasi tentang ganesha operation yang berisi visi dan misi, tujuan, alamat dan kontak.
![](http://i66.tinypic.com/mcpw68.jpg)

<b>Menu Bar Data</b><br/>
Menu Bar data dapat memilih atau melihat jadwal Mata Pelajaran, jadwal Try Out, Data Pelajar, Data Kerja, Data Tentor.<br/>

<h3>5.1.2 Deskripsi modul Sistem</h3>
<b>Form Login</b><br>
Form Login merupakan tampilan awal pada server atau sistem, disitu terdapat terdapat Link Forgot Password dan Capcha Lain. satu button yaitu Login untuk masuk sistem. ![](http://i68.tinypic.com/2rc4wvr.jpg)


<b>Dashbor Sistem</b><br/>
Dashbor Sistem untuk menambahkan dan mengedit user, menambahkan dan mengedit data, menambah dan mengedit jadwal, menambah dan mengedit informasi yang terdapat pada slide bar. button Logout untuk keluar dari Dashbor Sistem tersebut.


<b>Add User</b><br/>
Add User mengarahkan Sistem ke Form pendaftaran Pelajar, Tentor, Maupun Orangtua Pelajar. Admin dapan menginputkan data di form tersebut. terdapat button Daftar untuk menambahkan user. ![](http://i64.tinypic.com/mw5lir.jpg)


<b>Edit User</b><br/>
Edit User mengarahkan Sistem untuk mengedit user Pelajar, Tentor, maupun Orangtua Pelajar. Admin dapat mengedit informasi yang ada pada data tersebut, admin juga bisa menghapus dan kemudian menyimpannya. Disitu terdabat button Edit,Hapus dan Simpan. ![](http://i63.tinypic.com/212afz9.jpg)


<b>Edit dan Tambah Data</b><br/>
Sistem akan memilih mengedit atau menambah data di Jadwal Mata Pelajaran, Jadwal Try Out, Data Pelajar, Data Kelas, dan Data Tentor. ![](http://i66.tinypic.com/2qdcr3c.jpg)



<b>Edit Informasi</b><br/>
Sistem akan memasukan informasi lewat link dan akan ditampilkan ke Home User. di dalam Edit Informasi Sistem dapan menambahkan dan mengedit lnformasi, juga bisa menghapus informasi. ![](http://i63.tinypic.com/33tnfox.jpg)

<h3>5.2.2 Proses modul Sistem</h3>
<b>Form Login</b><br/>
Form Login merupakan tampilan awal pada server atau sistem, disitu terdapat terdapat Link Forgot Password. jika si admin menekan link forgot password, maka website akan mengarahkan admin ke "masukan email" untuk mendapat kode verifikasi dan jika si admin mengeklik Capcha Lain, maka website akan menampilkan kode capcha yang baru. satu button yaitu Login untuk masuk sistem. ![](http://i68.tinypic.com/2rc4wvr.jpg)<br/>


<b>Dashbor Sistem</b><br/>
Dashbor Sistem untuk menambahkan dan mengedit user, menambahkan dan mengedit data, menambah dan mengedit jadwal, menambah dan mengedit informasi yang terdapat pada slide bar. button Logout untuk keluar dari Dashbor Sistem tersebut.

<b>Edit User</b><br/>
